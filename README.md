# Junkboy

A junk drawer of an app.

## Database

### Migrations

Install `migrate` with the `sqlite3` tag:

```
go install -tags 'sqlite3' github.com/golang-migrate/migrate/v4/cmd/migrate@latest
```

Then run the migrations: `make migrations-up`.
